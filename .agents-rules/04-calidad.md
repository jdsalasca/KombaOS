Validación mínima
- Frontend: npm run lint, npm run test, npm run build.
- Backend: ./mvnw test.

Criterios
- Sin errores ni warnings críticos.
- Tests verdes antes de abrir PR.

Do
- Ejecuta comandos en el subproyecto correspondiente.
- Repite validaciones si hay cambios posteriores.

Don’t
- No ignores fallos de lint o tests.
